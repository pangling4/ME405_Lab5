Instantiates motor driver by initializing GPIO pins and turning the motor off for safety.

A timer object is also made. This will be used to send PWM'd power to the motors

Parameters
dir_PinPin object for setting motor direction: low() for fwd, high() for rev
speed_PinPin object for use with timer PWM channel to set motor speed
timer_NumAn integer designating which timer to use for the pin objects specified
channel_NumAn integer representing the channel used with the motor timer

This method sets the duty cycle to be sent to the motor to the given level.

Positive values cause torque in one direction, negative values in the opposite direction.

Parameters
dutyA signed integer representing desired duty cycle for the power sent to the motor
